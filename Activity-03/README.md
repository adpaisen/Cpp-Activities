# Temperature Converter

A simple C++ program that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Concepts Practiced

- `#include <iostream>` → Allows the program to use input/output features.
- `using namespace std;` → Lets you use `cout` and `cin` instead of writing `std::cout` and `std::cin`.
- `int main()` → The starting point of every C++ program.
- Functions → Used to separate conversion logic into reusable blocks.
- Parameters → Passes values into functions for processing.
- `switch statement` → Handles different conversion options selected by the user.
- Mathematical operations → Used for temperature conversion formulas.
- `return 0;` → Ends the program successfully.

## Features

- Celsius to Fahrenheit conversion
- Fahrenheit to Celsius conversion
- Celsius to Kelvin conversion
- Kelvin to Celsius conversion
- Menu-based user interaction

## Technologies Used

- C++
- Standard Input/Output Library (`iostream`)
