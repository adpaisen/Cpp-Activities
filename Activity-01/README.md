# Hello World

A simple C++ program that prints **"Hello, World!"** to the console.

## Concepts Practiced

- `#include <iostream>` → Allows the program to use input/output features.
- `using namespace std;` → Lets you use `cout` instead of writing `std::cout`.
- `int main()` → The starting point of every C++ program.
- `cout` → Prints text to the console.
- `return 0;` → Ends the program successfully.

## Technologies Used

- C++
- Standard Input/Output Library (`iostream`)
