# Calculator

A simple C++ calculator program that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Concepts Practiced

- `#include <iostream>` → Allows the program to use input/output features.
- `using namespace std;` → Lets you use `cout` and `cin` instead of writing `std::cout` and `std::cin`.
- `int main()` → The starting point of every C++ program.
- `cin` → Gets user input from the console.
- `cout` → Prints output to the console.
- `switch statement` → Handles different operations based on the user's selected operator.
- `return 0;` → Ends the program successfully.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero validation

## Technologies Used

- C++
- Standard Input/Output Library (`iostream`)
