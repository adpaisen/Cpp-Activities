# Quiz Game

A simple C++ console-based quiz game where players answer multiple-choice questions and receive a final score.

## Concepts Practiced

- #include <iostream> → Allows the program to use input and output features.
- #include <vector> → Provides a container for storing multiple quiz questions.
- using namespace std; → Lets you use cout and cin instead of std::cout and std::cin.
- struct → Groups related data such as questions, choices, and answers.
- vector → Stores quiz questions dynamically.
- for loop → Iterates through each question.
- if-else statement → Checks if the user's answer is correct.
- return 0; → Ends the program successfully.

## Features

- Multiple-choice questions
- Score tracking
- Answer validation
- Percentage calculation
- Console-based interaction

## Technologies Used

- C++
- Standard Input/Output Library
- Vector Library
